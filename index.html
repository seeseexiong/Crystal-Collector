Crystals Collector!

A random number is drawn. Below are four crystals, each with it's unique value. By clicking on a crystal you will add that crystal's value to the counter.
To win the game, you must add up your counter to match exactly to the random number. However, if your counter is greater than the random number then you lose. The value of the crystals are hidden. The value of each crystal changes for each game.
